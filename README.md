# 📊 HR Analytics Dashboard

## 📌 Project Overview

The HR Analytics Dashboard is an end-to-end Business Intelligence project designed to analyze employee data and provide workforce insights. The dashboard helps HR professionals monitor employee demographics, salary trends, departmental distribution, education levels, and attrition metrics.

Developed using Python, SQL Server, and Power BI.

---

## 🎯 Objectives

- Analyze workforce distribution across departments
- Monitor employee attrition and retention
- Evaluate salary trends by department
- Understand employee demographics and education levels
- Support data-driven HR decision-making

---

## 🛠️ Tools & Technologies

- Python (Pandas)
- SQL Server
- Power BI
- Jupyter Notebook

---

## 📊 Dashboard Features

### KPI Metrics

- Total Employees
- Active Employees
- Attrition Count
- Average Salary

### Interactive Visualizations

- Employee Distribution by Department
- Average Salary by Department
- Employee Distribution by Gender
- Employee Distribution by Education Level
- Department Filters
- Gender Filters
- Education Filters
- Attrition Filters

---

## 📈 Key Insights

- Total workforce consists of 1,000 employees
- Finance department has the highest employee count
- Operations department has the highest average salary
- Gender distribution is nearly balanced
- Education levels are evenly distributed across employees
- Attrition metrics help identify employee turnover patterns
